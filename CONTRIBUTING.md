# Contributing to Quantitative Strategy MCP

First off, thank you for considering contributing! It's people like you that make this project great.

## How Can I Contribute?

### Reporting Bugs
- Check if the bug has already been reported
- Include clear description and steps to reproduce
- Include Python version and OS information

### Suggesting Enhancements
- Open an issue with [Enhancement] tag
- Describe the feature and why it's useful
- Provide examples if possible

### Pull Requests
- Fork the repository
- Create a feature branch
- Make your changes
- Add tests if applicable
- Submit a pull request

## Development Setup

```bash
# Clone your fork
git clone https://github.com/YOUR_USERNAME/quantitative_strategy_mcp.git

# Install dependencies
pip install -r requirements.txt

# Run examples
python examples/usage_examples.py
```

## Code Style
- Follow PEP 8
- Use meaningful variable names
- Add docstrings to functions
- Keep functions focused and small

## Questions?
Feel free to open an issue for any questions!
